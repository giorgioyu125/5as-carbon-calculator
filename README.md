# 5as-carbon-calculator

## Descrizione
Questo progetto mira a un duplice obbiettivo, la produzione di
un calcolatore scolatisco della propria Impronta di CO2 e un analisi
dati personali, che pero' per ragioni di privacy verranno tenuti segreti.

**Nota importante**:
Dati basati sulle stime globali di Our World in Data (2018) e i dati del BAFA.
I dati restanti sono stati estratti da un altra libreria di nome [co2calculator](https://github.com/pledge4future/co2calculator)

## Tecnologie Utilizzate
- **Python 3.11+**

## Installazione
Lo script python e' completamente autosufficente, per cui puo' essere
avviato con:
```bash
python ./src/main.py
``` 
Trovate il sorgente delle slides direttamente in src/slides.
